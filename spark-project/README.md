# Apache Spark Data Processing Project

## Overview
This project demonstrates basic big data processing and analysis using Apache Spark and Python.

The project simulates data transformation and aggregation tasks commonly performed in data engineering and analytics workflows.

## Technologies Used
- Python
- Apache Spark (PySpark)
- Azure Databricks
- Data aggregation and transformation

## Features
- Data loading
- Filtering records
- Grouping and aggregation
- Summary statistics generation

## Purpose
This project was developed as part of my academic exposure to Big Data technologies while transitioning from QA Engineering into Data Science.
